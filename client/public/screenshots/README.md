# Screenshots Directory

This directory contains screenshots for the PWA install experience:

- `desktop.png` (1280x800) - Desktop view of the application
- `mobile.png` (390x844) - Mobile view of the application

## Requirements:
1. Images must match the exact dimensions specified
2. Screenshots should show the app in use with real tasks
3. Both light and dark mode variants recommended

To update screenshots:
1. Use Chrome DevTools device toolbar for mobile screenshot
2. Capture full desktop view for wide screenshot
3. Ensure the app shows relevant content and features
